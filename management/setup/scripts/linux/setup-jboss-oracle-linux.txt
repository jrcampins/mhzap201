--------------------------------------------------------------------------------
Instalación de mhzap201 1.0 en Windows, con JBoss y Oracle
--------------------------------------------------------------------------------

01. Descargar e instalar Oracle 11g R2 Express Edition.

    NOTA: el link de descarga es http://www.oracle.com/technetwork/database/enterprise-edition/downloads/index.html

    Opcionalmente, instalar Oracle SQL Developer 3.2.2 (3.2.20.09.87).

    NOTA: el link de descarga es http://www.oracle.com/technetwork/developer-tools/sql-developer/downloads/index.html

    Establezca la variable de entorno ORACLE_HOME con la ruta del directorio donde instaló Oracle.


02. Descargar e instalar JDK 1.7.

    NOTA: el link de descarga es http://www.oracle.com/technetwork/java/javase/downloads/java-se-jdk-7-download-432154.html	

    Establezca la variable de entorno JAVA_HOME con la ruta del directorio donde instaló el JDK.


03. Descargar JBoss AS 7.1.1 Final.

    NOTA: el link de descarga es http://www.jboss.org/jbossas/downloads/

    Para instalar basta con extraer la carpeta comprimida en un directorio. Es recomendable instalar en el directorio /opt/jboss

    Establezca la variable de entorno JBOSS_HOME con la ruta del directorio donde instaló JBoss.

    NOTA: si no instaló JBoss en /opt/jboss, puede crear el enlace simbólico /opt/jboss para JBOSS_HOME

        ln -s $JBOSS_HOME /opt/jboss


04. Crear el directorio mhzap201, preferiblemente en /opt

    Establezca la variable de entorno MHZAP201_HOME con la ruta del directorio mhzap201.

    NOTA: si no creó el directorio mhzap201 en /opt, puede crear el enlace simbólico /opt/mhzap201 para MHZAP201_HOME

        ln -s $MHZAP201_HOME /opt/mhzap201


05. Descomprimir el archivo zip VnnRaammdd en /opt/mhzap201

    NOTA: esta acción crea el directorio /opt/mhzap201/VnnRaammdd


06. Crear y otorgar privilegios al usuario MHZ en Oracle.

    Conectar con el usuario SYS:

        connect SYS/password as SYSDBA; 

        NOTA: el password es definido al instalar Oracle.

    Crear el usuario MHZ:
        
        create user MHZ identified by sesamo;

    Otorgar privilegios al usuario MHZ:
        
        grant connect, DBA to MHZ;


07. Crear la base de datos en Oracle.

    Conectar con el usuario MHZ:

        connect MHZ/sesamo;

    Ejecutar el script makedb.sql que se encuentra en /opt/mhzap201/VnnRaammdd/setup/oracle/windows:

        spool /opt/mhzap201/VnnRaammdd/setup/oracle/windows/makedb.log
        @/opt/mhzap201/VnnRaammdd/setup/oracle/windows/makedb

    Comprobar el resultado de la ejecución (revisar el archivo /opt/mhzap201/VnnRaammdd/setup/oracle/windows/makedb.log).

08. Configurar JBoss (seguir los pasos indicados en el archivo /opt/mhzap201/VnnRaammdd/configuracion-jboss-windows.txt).

09. Comprobar el funcionamiento de la aplicación.

    NOTA: el link de la aplicación es http://localhost:8080/mhzap201-web/
