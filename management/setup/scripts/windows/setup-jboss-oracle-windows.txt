--------------------------------------------------------------------------------
Instalación de mhzap201 en Windows, con JBoss y Oracle
--------------------------------------------------------------------------------
01. Crear el directorio mhzap201, preferiblemente en C:\

    El path del directorio NO debe contener espacios ni caracteres especiales
    Si no creó el directorio mhzap201 en C:\, puede crear el enlace simbólico C:\mhzap201
    http://www.howtogeek.com/howto/16226/complete-guide-to-symbolic-links-symlinks-on-windows-or-linux/
    describe como crear enlaces simbólicos.


02. Descomprimir el archivo zip VnnRaammdd en C:\mhzap201

    Esta acción crea el directorio C:\mhzap201\VnnRaammdd


03. Ajustar las variables de entorno en:

    C:\mhzap201\variables-conf.bat
    C:\mhzap201\variables-dbdc.bat
    C:\mhzap201\variables-home.bat


04. Crear y otorgar privilegios al usuario MHZ en Oracle.

    Conectar con el usuario SYS:

        connect SYS/password as SYSDBA; 

        NOTA: el password de SYS es definido al instalar Oracle.

    Crear el usuario MHZ:
        
        create user MHZ identified by sesamo;

    Otorgar privilegios al usuario MHZ:
        
        grant connect, DBA to MHZ;


05. Crear la base de datos en Oracle.

    Conectar con el usuario MHZ:

        connect MHZ/sesamo;

    Ejecutar el script makedb.sql que se encuentra en C:\mhzap201\VnnRaammdd\setup\oracle\windows:

        spool C:\mhzap201\VnnRaammdd\setup\oracle\windows\makedb.log
        @C:\mhzap201\VnnRaammdd\setup\oracle\windows\makedb
        spool off

    Comprobar el resultado de la ejecución (revisar el archivo C:\mhzap201\VnnRaammdd\setup\oracle\windows\makedb.log).

06. Configurar JBoss (seguir los pasos indicados en el archivo C:\mhzap201\VnnRaammdd\configuracion-jboss-windows.txt).
