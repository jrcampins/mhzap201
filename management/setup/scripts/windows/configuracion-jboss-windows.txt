--------------------------------------------------------------------------------
Configuración de JBoss en Windows
--------------------------------------------------------------------------------

01. Cambiar 8080 por 8082 en los archivos xml de jboss (12 veces en 8 archivos)

        standalone-hornetq-colocated.xml
        standalone-jts.xml
        standalone-xts.xml
        domain.xml
        standalone-full-ha.xml
        standalone-full.xml
        standalone-ha.xml

    <datasources>

    Por ultimo, agregar <sso reauthenticate="false"/> a la definicion de default-host:

    <subsystem xmlns="urn:jboss:domain:web:1.1" default-virtual-server="default-host" native="false">
        <connector name="http" protocol="HTTP/1.1" scheme="http" socket-binding="http"/>
        <virtual-server name="default-host" enable-welcome-root="true">
            <alias name="localhost"/>
            <alias name="example.com"/>
            <sso reauthenticate="false"/>
        </virtual-server>
    </subsystem>


03. Ajustar parametros de la JVM

    Modificar -Xms, -Xmx y -XX:MaxPermSize
    
    Agregar -Dorg.apache.el.parser.COERCE_TO_ZERO=false

    En C:\jboss\bin\standalone.conf.bat, donde dice:

        rem # JVM memory allocation pool parameters - modify as appropriate.
        set "JAVA_OPTS=-Xms64M -Xmx512M -XX:MaxPermSize=256M"

    Colocar:

        set "JAVA_OPTS=-Xms128M -Xmx1024M -XX:MaxPermSize=384M -Dorg.apache.el.parser.COERCE_TO_ZERO=false"


04. Copiar el directorio C:\mhzap201\VnnRaammdd\setup\jboss\modules\com\oracle a C:\jboss\modules\com    

05. Copiar el directorio C:\mhzap201\VnnRaammdd\setup\jboss\welcome-content\mhzap201 a C:\jboss\welcome-content

06. Copiar el directorio C:\mhzap201\VnnRaammdd\setup\jboss\resources\windows a C:\jboss

07. Ajustar las variables en C:\jboss\resources\mhzap201.properties

    Especifique la ruta completa de la carpeta resources de mhzap201 en la propiedad DIR.
    Habilite y especifique la ruta completa de la carpeta welcome-content de JBoss en la propiedad DOC.
    Habilite y especifique las propiedades JDBC que corresponden a Oracle.
    
    DIR=C:\\mhzap201\\VnnRaammdd\\resources
    #
    # JBoss
    #
    DOC=C:\\jboss\\welcome-content
    #
    # Oracle
    #
    JDBC_DRIVER=oracle.jdbc.OracleDriver
    JDBC_URL=jdbc:oracle:thin:@localhost:1521:XE
    JDBC_USER=MHZ
    JDBC_PASSWORD=sesamo

    NOTA: Los valores predeterminados coinciden con los valores predeterminados de instalación.
    NOTA: el separador de las rutas (DIR y DOC) tiene que ser doble backslash (\\).
    

08. Ejecutar el script C:\mhzap201\VnnRaammdd\setup\jboss\standalone\scripts\windows\standalone-start.bat

09. Ejecutar el script C:\mhzap201\VnnRaammdd\setup\jboss\standalone\scripts\windows\ear-deploy.bat
